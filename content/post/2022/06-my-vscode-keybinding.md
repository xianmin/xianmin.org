## 移动相关

| 快捷键  | 说明       | 补充                        |
|---------|----------|-----------------------------|
| `C-f`   | 下一个字符 | 记忆： f - forward           |
| `C-b`   | 上一个字符 | 记忆： b - backward          |
| `C-n`   | 下一行     | 记忆： n - next              |
| `C-p`   | 上一行     | 记忆： p - previous          |
| `C-a`   | 行首       | 记忆： a - 第一个字母        |
| `C-e`   | 行尾       | 记忆: e - end               |
| `M-f`   | 下一个词   | VSCode 中文断词设置，见附录A |
| `M-b`   | 上一个词   |                             |
| `C-M-f` | 上一段     |                             |
| `C-M-b` | 下一段     |                             |
| `C-M-a` | 页首       |                             |
| `C-M-e` | 页尾       |                             |

## 附录A： Word seperator for Chiense (中文标点断句)
在 VSCode 中修改如下 setting ：

```
"editor.wordSeparators": "`~!@#$%^&*()-=+[{]}\\|;:'\",.<>/?、，。？！“”‘’；：（）「」【】〔〕『』〖〗",
```
